# Kanban Board

This directory contains tasks to populate a Kanban board.
It can be rendered using the [markban tool](https://github.com/dswisher/markban).

## Colors

I am going to try an use colors to differentiate the sub-project to which the task applies, as follows:

- `red` - bugs, regardless of sub-project
- `yellow` - sprites and sprite tool
- `green` - ECS
- `blue` - TBD
- `magenta` - TBD
